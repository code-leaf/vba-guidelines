# VBAコード生成と品質改善のためのガイドライン集

このリポジトリは、VBAコード生成と品質改善のためのガイドライン集です。

## 📁 ファイル一覧

- **VBAコード生成ガイドライン.md**: 0からVBAコードを生成する際の指示書
- **VBAコードブラッシュアップ指示書.md**: 既存VBAコードを改善する際の指示書

## 📖 使い方

1. 各ガイドラインファイルの内容を各種生成AI（Claude.aiのプロジェクトなど）に登録
2. プロジェクト内で会話することで、ガイドラインに従ったコードが生成されます
